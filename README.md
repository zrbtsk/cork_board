# проект в разработке
Cork_board - pet-проект, пробковая доска для заметок.

## наброски функционала:

[] передвижение заметок по доске
[] увеличение и уменьшение до определенных велечин
[] при нажатии на заметку выскакивает меню с возможностью увеличивать уменьшать, менять дизайн и просто писать в заметке
[] одно нажатие - перемещение, два нажатия - открывается меню 
[] пабота с текстом
